# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `@enkryptify/sdk`, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email us at **security@enkryptify.com** with:

- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant logs or screenshots

We will acknowledge your report within 48 hours and aim to provide a fix within 7 days for critical issues.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |

## Disclosure Policy

We follow coordinated disclosure. We ask that you give us reasonable time to address the vulnerability before making any information public.
